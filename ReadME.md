# Magic-Paint.js API Documentation

Magic-Paint.js の開発者向けドキュメントへようこそ。

このディレクトリには、Magic-Paint.js の API、MOD 開発、SDK、AI フレームワークなど、拡張開発に必要なドキュメントがまとめられています。

---

## 📚 ドキュメント一覧

| ファイル | 説明 |
|----------|------|
| **api.md** | Magic-Paint.js が提供する公開APIの概要と基本的な使い方です。 |
| **MOD_API_REFERENCE.md** | MOD API のリファレンスです。各関数・オブジェクト・イベントの詳細仕様を掲載しています。 |
| **MOD_DEVELOPMENT_GUIDE.md** | MOD 開発の手順をまとめたガイドです。開発環境の準備から公開までを解説しています。 |
| **MOD_SDK.md** | Magic-Paint.js SDK の導入方法やサンプルコードを掲載しています。 |
| **FEATURE_PACK.md** | Feature Pack（機能拡張パック）の仕様や作成方法について説明しています。 |
| **AI_FRAMEWORK.md** | AI Framework の仕様やAIアシスタントとの連携方法について説明しています。 |

---

## 🎯 このドキュメントの対象者

このドキュメントは以下のような方を対象としています。

- Magic-Paint.js のMODを開発したい
- 独自のツールやブラシを追加したい
- Feature Pack を作成したい
- AI機能を利用したい
- Magic-Paint.js の内部APIを利用したい

JavaScript の基本的な知識があることを推奨します。

---

## 🔄 バージョンについて

このドキュメントは、Magic-Paint.js の開発状況に合わせて更新されます。

APIやSDKの仕様は、将来のバージョンで変更・追加される場合があります。
MOD.jsonは現在はmanifest.jsonです
